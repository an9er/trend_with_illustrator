# trend_with_illustrator
# 将通达信中的走势图重新绘制到Adobe Illustrator上  
## 大体思路 
从通达信获取5M走势数据，数据包含字段如下：  

      日期	    时间	    开盘	    最高	    最低	    收盘	    成交量	    成交额
      
Illustrator 支持js，通过js读取数据，绘制到Illustrator上。
